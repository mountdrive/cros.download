RMA Shims

An RMA shim image is a combination of Chrome OS factory bundle components. When flashed to a USB drive, it is a useful tool for performing diagnostics and repairs.

See the ChromiumOS documentation for more information on how they work.

This webpage contains download links for all publicly available RMA shims for various Chrome OS devices.
